# Iaac - Terraform
Infraestrutura as Code da arquitetura de referência

# Principais Comandos

# Iniciar o terraform
terraform init

# Validar o terraform 
terraform validate

# Mostra as alterações necessárias para a configuração atual
terraform plan

# Criar a Infraestrutura
terraform apply

# Derrubar a Infraestrutura provisóriamente
terraform destroy
